📖 Simultaneous_Interpreter 使い方

NEW: プロンプターモードと高さ調整

  • プロンプターモード: 右端のボタンでON。暫定表示がログと同じ特大サイズになります。枠（青い線）に文字が届くと自動でログに確定されます。
  • 高さの変更: 暫定エリアの下にある青い線をドラッグすると、エリアの高さを変えられます。初期状態は自動伸縮です。

1. 基本的な使用手順

  • 設定メニュー: 上部バーを横にスクロールすることですべての設定メニューが表示されます。
  • 事前設定: 上部バーで「音声（話す側）」と「翻訳（表示側）」を選択。デフォルトは日本語→中国語(簡体)です。
  • 開始: 青色の「同時翻訳開始」ボタンをクリックし、ブラウザのマイクの使用を許可してください。
  • 発話: 画面に向かって話すと、上部の青枠（暫定ボックス）に文字が溜まり、無音を検知すると下部に確定ログとして出力されます。
  • 暫定訳機能: 「暫定訳」にチェックを入れると、確定前の青枠内でもリアルタイムに翻訳結果を表示します。
  • 終了・保存: 「ログ名:」を入力している場合、「停止」ボタンを押すと、ログがテキストファイルとして自動保存されます。

2. 便利な操作・キャンセル機能

  • 言語のクイック入れ替え: 上部バーの「⇄」ボタンを押すと、言語設定を瞬時に入れ替えることができます。双方向のやり取りに便利です。
  • 逆翻訳による精度確認: 「逆翻訳」にチェックを入れると、翻訳結果をさらに自分の言語へ訳し戻して表示します。戻ってきた文が不自然な場合は、言い方を変えて再度発話することで、誤解を防ぐことができます。
  • リアルタイム配色: 「色」を変更すると、確定済みのログだけでなく、現在入力中の「暫定訳」の色も即座に反映されます。
  • 発話の強制確定（即時ログ出力）: 無音（分割秒数）を待たずに、現在の入力をすぐにログへ出力したい時は、上部の青枠を「左クリック」、「1本指でタップ」、またはキーボードの「Enterキー」を押してください。
  • 発話のリセット（キャンセル）: 言い間違えた時は、確定前に上部の青枠を「右クリック」、「2本指でタップ」、または「Escキー」を押してください。
  • 音声読み上げ: 確定ログの「翻訳文（大きな文字）」を直接タップするか、横の「🔊」ボタンを押すと読み上げます。
  • 割り込み再生: 読み上げ中に別のログをタップすると、前の再生を中断して新しい文章をすぐに読み上げます。
  • スライダーによる動的調整: 「大」スライダーを動かすと、メイン翻訳だけでなく上部の原文サイズも自動で最適化（メインの40%）され、常にバランス良く表示されます。

3. セキュリティと高度な文字起こし活用

  • 高セキュリティ文字起こし: 「音声」と「翻訳」を同じ言語（例：日本→日本）に設定した場合、外部の翻訳APIサーバーへテキストを送信する処理が完全にスキップされます。字幕が1段だけの「文字起こしモード」になり、画面を広く使えます。
  • 情報の保護: 同言語設定時は翻訳通信が発生しないため、機密性の高い会議の議事録作成ツールとして安全かつ強力に機能します。

4. 利用上の注意点と対応ブラウザ（重要）

  • ◎ 動作確認済み: Google Chrome / Microsoft Edge
  • △ 条件付き動作: Safari (iPhone/Mac)
  • × 非対応: Vivaldi / Brave / Firefox等
